# Installation
After installing the dependencies, lets install the bot:
```bash
 git clone https://github.com/CerNerCompany/Self-Bot
 cd S*
 chmod +x C
 ./C install
 ./C config
 ./C login-Cli # Will ask you for a phone number & confirmation code.
 ./C start
 ---------
 screen ./Co*
 ---------
```
